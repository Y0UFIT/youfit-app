## Branch
### Branch Type
- v* : 최종 버전
- hot-* : 핫픽스
- dev : 중심 기능
- feat-* : 새 기능

</br>

## Commit
### Commit Type
- feat: 새 기능 추가
- refactor: 코드 리팩토링
- fix: 버그, 기능 수정
- remove: 코드 또는 파일 삭제
- style: 코드 스타일 변경
- resource: 리소스 추가
- test: 테스트 코드 추가
- chore: 기타 (코드 수정X)

### Subject
- 커밋 타입은 대문자, 이후 소문자 작성 (한/영 혼합 작성 가능)
